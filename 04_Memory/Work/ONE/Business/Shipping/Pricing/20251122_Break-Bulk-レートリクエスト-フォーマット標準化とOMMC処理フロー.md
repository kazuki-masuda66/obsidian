# Break Bulkレートリクエスト-フォーマット標準化とOMMC処理フロー

## 概要
Break Bulk貨物のレートリクエストに関する標準化フォーマットとOMMC（ONE Manila Marketing & Commercial Center）での処理フローについてのメールスレッドから抽出したノウハウ。

## 内容

### Break Bulkリクエストの標準化フォーマット

#### メール件名の標準化
**フォーマット**: `Office Code / YYYYMMDD / Sales Initials / Sequence Number / Customer / Port Pair`

**例**: `SINBB / 20250722 / LHL / 001 / AAA Shipping Pte Ltd / SGSIN - USORF`

#### 必須情報
1. **Excelテンプレートの提出**
   - テンプレートURL: https://bacs.apps.one-line.com/bacs/cargo_policies/113/1355
   - BB Tabに記入
   - **重要**: OPS Teamの評価に必要で、このファイルなしでは進められない

2. **貨物写真・図面**
   - COG（Center of Gravity）とリフティングポイントの記載が必要

### ノウハウ

#### Break Bulkリクエスト時の注意点
1. **メール件名の標準化は必須**
   - 標準化されていないと、OMMCから修正を求められる
   - フォーマットに従うことで処理がスムーズになる

2. **Excelテンプレートは必須**
   - OPS Teamの評価に必要
   - テンプレートなしでは進められない

3. **貨物情報の詳細が必要**
   - 寸法、重量、COG、リフティングポイント
   - 写真・図面があると評価がスムーズ

4. **正しいPICへのルーティング**
   - Break BulkはOMMCが担当
   - 不明な場合は、GHQに確認して正しいPICを追加してもらう
