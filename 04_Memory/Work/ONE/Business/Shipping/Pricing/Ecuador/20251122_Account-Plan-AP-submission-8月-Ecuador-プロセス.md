# Account Plan (AP) submission - 8月 Ecuador プロセス

## 概要
LWE（Latin West Ecuador）向けの8月Account Plan提出プロセス。Week 32-36のVessel CoverageとBSA配分の詳細。

## 背景情報

### AP提出サイクル
- **配布日**: 2025年6月30日（MKTGからOrigin officesへ配布）
- **提出期限**: 2025年7月10日 COB（Salesから返送）
- **最終確定**: 2025年7月15日（MKTGが最終化して全オフィスへ配布）

### Vessel Coverage（Week 32-36）
- **Week 32**: 
  - AX1: HUMBOLDT EXPRESS V:HBET2532E ETA YTN 05/08/2025
  - AX2: ONE SPIRIT V:OITT2531E ETA SHA 12/08/2025
  - AX3: VANTAGE V:VTDT2532E ETA SHA 17/08/2025
  - AX4: NAVIOS LAPIS V:NLVT0007E ETA NGB 04/08/2025

### BSA配分ルール
- **8月のBSA**: Vessel sizeに応じてratioed（按分）
- **大型船到着時**: 3月末に大型船が到着する場合、サービスごとの配分が増加する可能性

## プロセス詳細

### 1. AP Input Fileのアクセス
- **場所**: Shared Drives >> LW AP >> "AP Input File"
- **Google Sheet**: https://docs.google.com/spreadsheets/d/1cXbUAfG3dkXxiCyvClF2u9z-u-KbeZSV4lVdIbI_hcw/edit
- **前月のテンプレート**: 既にリリース済み（Origin officesがレビュー・調整可能）

### 2. 各オフィスのAP配分確認
- **確認方法**: 個別FO_Input working AP file >> "AP Summary"タブ
- **表示内容**: 
  - Total allocation（総配分）
  - Service lane別の配分

### 3. 提出時の注意事項

#### 必須チェック項目
- **PIC Check列**: 担当者が内容を確認し、"Done"を入力
- **Completeチェックボックス**: 全てのAPをチェック後、必ず"Complete"にチェック
  - **重要**: "Complete"にチェックしないと提出がキャプチャされない

#### 変更不可ルール
- **配布後は変更不可**: APが配布された後は、変更は一切受け付けない
- **提出前に最終確認**: 提出前に必ず内容を確認し、最終版であることを確認

## 実務ノウハウ

### 提出前の最終チェックリスト
- [ ] 全てのAPファイルの内容を確認
- [ ] PIC Check列に"Done"を入力
- [ ] "Complete"チェックボックスにチェック
- [ ] 提出期限（7月10日COB）を確認

### よくある問題と対策
- **問題**: "Complete"にチェックし忘れて提出がキャプチャされない
  - **対策**: 提出前に必ず"Complete"チェックボックスの状態を確認

- **問題**: AP配布後に変更が必要になった
  - **対策**: 配布前に必ず最終確認。変更は受け付けられないため、事前に十分な確認が必要
