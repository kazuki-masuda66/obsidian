# AFLA-BBとかの意味-UN-LOC-CODEとOffice-Codeの違い

## 概要
BBとかの意味に関するメモ。UN loc（と昔は言っていた）、世界共通のUN LOCATION CODE、TYOBBやSELBBといった我々のオフィスコード、Business BrunchかBooking Brunchの略、GHQおよびRHQ、SINHOとかRICHO、Head Ofiice、SINBBやRICBB、一部BAもありますが、このAはおそらくAgent（ONEの支店・現法以外）、SMZBA（清水の営業拠点は鈴与さんとのジョイント代理店）、輸送経路およびBL作成などでは世界共通のUNLOCの使用が必須、Bookingを受ける・顧客の担当拠点など社内で識別するのにONE独自のOffice codeを使用しているについて記録されています。
