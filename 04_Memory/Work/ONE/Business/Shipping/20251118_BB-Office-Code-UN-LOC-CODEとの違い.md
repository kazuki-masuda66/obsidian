# BB-Office-CodeとUN-LOC-CODEの違い

## 概要
BB（Business Branch/Booking Branch）などのOffice CodeとUN LOC CODE（UN Location Code）の違いについて説明します。船社が輸送するルートで使用するUN LOC CODEと、社内の拠点管理で使用するOffice Codeの違いを記載しています。

## UN LOC CODE（UN Location Code）

### 定義
- **世界共通のUN LOCATION CODE**
- 世界中の都市に定められている（我々もDoor deliveryの際などに都市名のコード使用）
- 基本的には輸出入のポイントとなる空港や港や鉄道の駅にも定められている

### 使用場面
- **輸送経路およびBL作成などでは世界共通のUNLOCの使用が必須**
- 国際的な輸送ルートの識別に使用

## Office Code（社内の拠点管理コード）

### 定義
- TYOBBやSELBBといった我々のオフィスコード
- これはあくまでも社内の拠点管理のコード
- おそらく**Business BranchかBooking Branchの略**

### 使用場面
- 社内での管理に重要で、どこの拠点の担当している顧客なのか、どこでBookingをうけたか＝BLを発行するのか、入金受けるのかなどでの管理となります
- **Bookingをした場所、BLの発行場所、お金の支払い地が異なるケースもあります**

## GHQおよびRHQ
- GHQおよびRHQは、SINHOとかRICHOと、Head Office（支店BBではない）と識別されています
- 実際にSingaporeにもRichmondにも営業もいるので、SINBBやRICBBも存在します

## BA（Agent）
- 一部BAもありますが、このAはおそらくAgent（ONEの支店・現法以外）
- 日本だとSMZBA（清水の営業拠点は鈴与さんとのジョイント代理店）とかになっています

## まとめ

| 項目 | UN LOC CODE | Office Code |
|------|------------|-------------|
| **用途** | 国際的な輸送ルートの識別 | 社内の拠点管理 |
| **使用場面** | BL作成、輸送経路 | Booking管理、顧客担当拠点 |
| **必須性** | 国際輸送では必須 | 社内管理用 |
| **例** | JPTYO（東京）、USLAX（ロサンゼルス） | TYOBB（東京支店）、SELBB（ソウル支店） |

- **輸送経路およびBL作成などでは世界共通のUNLOCの使用が必須**
- **Bookingを受ける・顧客の担当拠点など社内で識別するのにONE独自のOffice codeを使用している**
