## 概要
LATAM（Latin America）リージョン向けのEagle Xトレーニング準備に関するノウハウ。テスト環境でのサンプル作成、Route Portfolio Simulationの準備、トレーニング対象アカウントの選定プロセスをまとめた。

## トレーニングの目的

### 対象国
- **前年未参加国**: 前年にトレーニングに参加しなかった国々
- **目的**: Eagle Xの使用方法を習得し、テンダー処理を効率化

### トレーニング内容
1. **Route Portfolio Simulation**: ルートポートフォリオのシミュレーション方法
2. **Pricing Tool**: Pricing Toolの使用方法
3. **Game Plan Summary**: 将来的にGame Plan Summaryも含める予定（全リージョン同時実施）

## テスト環境でのサンプル作成

### サンプルテンダーの選定
- **テンダー名**: LKA G-BR105855 BR105855 FRIGOL S.A.
- **選定理由**: LATAM向けのトレーニングに適したテンダー

### テスト環境への追加
- **ステータス**: Test Status Listに追加済み
- **目的**: トレーニングで使用するサンプルとして利用可能

## Route Portfolio Simulationの準備

### シミュレーション期間の確認
- **契約開始日**: 2024年1月
- **利用可能データ**: 2024年1月のLifting結果のみ
- **提案期間**: 2023年1月〜12月のデータを使用することを提案

### シミュレーションの目的
- **ルートポートフォリオ分析**: 各ルートのボリュームと収益性を分析
- **プレゼンテーション資料**: トレーニング用のプレゼンテーション資料を作成

## Game Plan Summaryの展開計画

### 現時点の状況
- **トレーニング対象外**: 現時点ではRoute PortfolioとPricing Toolのみ
- **理由**: Game Planトレーニングは全リージョン同時に実施する必要がある

### 今後の展開
- **GCSMの進捗に依存**: GCSMのGame Plan開発の進捗に応じて、全リージョン向けのトレーニングを計画
- **実施時期**: 未定（GCSMの進捗次第）

## 実務上の注意点

### テスト環境の準備
- **サンプルテンダーの追加**: トレーニング用のサンプルテンダーをTest Status Listに追加
- **データの確認**: シミュレーションに使用するデータの整合性を確認

### Route Portfolio Simulation
- **期間の選定**: 利用可能なデータに基づいてシミュレーション期間を選定

### トレーニングの実施
