# WNS India group address - 設定とメンバー管理

## 概要
WNS Indiaグループアドレスの設定とメンバー管理プロセス。Delegated mailboxとの違いと、Manager roleによるメンバー管理。

## 背景情報

### グループアドレス設定
- **目的**: WNS Indiaチームのアクセス権限設定
- **設定者**: Kazuki Masuda
- **Manager role**: SivaとMuraliもManager roleに設定

### グループアドレスとDelegated Mailboxの違い
- **Delegated Mailbox**: パスワード/2FAが必要
- **Group Address**: ユーザーを追加するだけでアクセス可能、2FA不要
- **Email Subscription**: Group Addressには"Email" subscriptionが設定されており、グループ内の全ユーザーが個人アカウントでメールを受信

- **Email Subscription**: グループ内の全ユーザーが個人アカウントでメールを受信
- **2FA不要**: Delegated mailboxと異なり、2FAは不要
- **Manager role**: Manager roleのメンバーが自由にメンバーを追加・削除可能
