# Eagle X-Status list clean up-FY22、FY23、FY24

## 概要
Eagle XプロジェクトのStatus list clean upに関するメモ。FY22、FY23、FY24のStatus listの整理方法、フォルダ名の変更、アカウント追加ツール、JPの11月の対象チェック、EUAとLAのStatus listについて記録しています。

## 内容

### Status list clean upの方針
- きれいにはするものの、最終的にすべてのアカウントがTESTリストに載せられる。LIVEリストはその中かからCandidateになったものがLIVEに載せられる

### フォルダ名は変更可能
- **FY22→過去のアーカイブ**: 過去のConversion studyの結果が見れなくなったらこまるからStatuslistもアーカイブする
- **FY23→FY24に受領予定のTenderのConversion studyをやり直す**:
  - つまりFY24向けにこれから受領予定のファイルですでにConversion studyをしているものは残す
    - JP、NAでCSTが今Conversion studyしているもの
    - SAS・EASで直近Conversion studyしているもの
    - LAとEUAの直近10月に来るTenderたちでConversion studyしているもの
  - LAとEUAはいったんリセットしてリストを再整理してUploadし直す

### アカウント追加ツールは追加は可能
- ただし、削除や編集はまだPWCしかできないから都度連絡が必要
