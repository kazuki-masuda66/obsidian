# FY24 Target (Eagle X) - 目標設定と管理ノウハウ

## 概要

FY24 Eagle Xプロジェクトの目標設定について。各ターゲットに対して定量的な目標を設定し、SuccessFactorでの個人目標設定と提出を実施。

## 背景

- **日付**: 2024年5月23日
- **発信者**: Koji Nakai (GHQ Eagle X)
- **受信者**: Eagle X Team全員
- **目的**: FY24 Eagle Xプロジェクトの目標設定と、個人目標の設定・提出

## 主要ポイント

### 1. 目標設定のアプローチ

- **定量的目標**: 各ターゲットに対して、可能な限り定量的な目標を設定
- **挑戦的な目標**: 一部の目標は、より挑戦的で野心的な目標となっている
- **個人目標**: SuccessFactorで個人目標を設定・提出

### 2. 目標設定のプロセス

- **目標の共有**: FY24 Target for Eagle Xを添付ファイルで共有
- **個人目標設定**: 各メンバーがSuccessFactorで個人目標を設定
- **提出期限**: 指定された期限までに個人目標を提出

### 3. 目標の特徴

- **定量的**: 可能な限り定量的な目標を設定
- **挑戦的**: 一部の目標は、より挑戦的で野心的
- **達成可能**: 達成可能な目標を設定

## 実務上の注意点

### 1. 個人目標の設定

- **目標の確認**: FY24 Target for Eagle Xを確認し、個人目標を設定
- **定量的目標**: 可能な限り定量的な目標を設定
- **提出期限**: 指定された期限までに提出

### 2. 目標の管理

- **進捗確認**: 定期的に目標の進捗を確認
- **達成状況**: 目標の達成状況を記録
- **フィードバック**: 目標達成に向けたフィードバックを実施

### 3. SuccessFactorの活用

- **個人目標設定**: SuccessFactorで個人目標を設定
- **進捗管理**: SuccessFactorで進捗を管理
- **評価**: SuccessFactorで評価を実施
