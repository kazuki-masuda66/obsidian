# Eagle-X new resource introduction - 夜勤対応とリソース管理

## 概要
Eagle-Xプロジェクトへの新リソース導入と夜勤対応のアレンジメント。5名の新メンバー追加と、Day-time/Night-shiftのシフト管理。

## 背景情報

### 新メンバー導入
- **新メンバー数**: 5名
- **総メンバー数**: 12名（新メンバー追加後）
- **学習期間**: 2ヶ月間（通常勤務時間で既存メンバーから学習）
- **シフト開始**: 学習期間終了後

### 夜勤対応開始
- **開始時期**: 2024年11月から
- **目的**: 24時間対応体制の構築
- **リソース**: Day-time 2名、Night-shift 2名

## シフト管理

### 勤務時間
- **Day-time**: 8:30 - 17:30 VNT（ベトナム時間）
- **Night-shift**: 15:00 - 24:00 VNT（ベトナム時間）
- **リソース**: Day-time 2名、Night-shift 2名
- **ローテーション**: 週次ローテーション（状況に応じて変更可能）

### シフトアレンジメント詳細
- **詳細スケジュール**: Google Sheetsで管理
- **URL**: https://docs.google.com/spreadsheets/d/1rBqsWPJ_g_uBKr3lHqqqz3we_WEHZjb3_r75DlWVcVI/edit?hl=en&forcehl=1&gid=0#gid=0

### シフト延長
- **延長期間**: 2025年4月11日まで
- **リクエスト者**: KaWah Lau
- **承認**: Neo (Nam) Nguyenが承認、HRに通知

## 実務ノウハウ

### 新メンバー導入プロセス
1. **紹介**: Slide introductionとMembers profileを共有
2. **学習期間**: 2ヶ月間の学習期間を設定
3. **シフト開始**: 学習期間終了後にシフト対応開始

### シフト管理のポイント
- **週次ローテーション**: メンバーが週次でローテーション
- **柔軟性**: 状況に応じてルールを変更可能
- **24時間対応**: Day-timeとNight-shiftで24時間対応体制を構築

### リソース管理
- **Day-time**: 2名で対応
- **Night-shift**: 2名で対応
- **総リソース**: 12名（新メンバー5名含む）
